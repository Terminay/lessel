# Listener Bot 🤖
A deployed backend bot for reading Discord Details, can be amended as per Discord Needs.
---

## Features

- Able to Read Quiz Messages
- Able to transition those messages to a server
- Whole pipeline is based on this bot

## How to Use?

1. Clone Repo
2. Check Schema as per your quiz bot
3. Amend Env as per your server
4. Goto Discord console and get the key
5. Run it (commands given in package)

Now all the captured quiz (or selected) messages with the correct schema will be sent to a server deployed on a webapp, this will create a custom leaderboard (or any use case)

## Use Cases

- Wanting to Deploy Bot LeaderBoard on a site
- Wanting to retrieve certain messages from a certain person or people
- Tracking group events
- Centralizing your discord, whatsapp and Slack
- Any use case related to Discord

Usage shall not destroy the freedom of others.
